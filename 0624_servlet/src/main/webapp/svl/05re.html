POST 방식 전송 테스트




GET 방식 전송 테스트




form element : 화면에서 input type들을 통해 받은 데이터를 서버로 보내기 위해 묶는 객체

form action : 데이터를 보낼 주소를 넣는곳

method : 데이터를 보내는 방법을 정의

method = get방식 : 주소창에 전송하는 데이터들이 모두 보이는 방식
보안이 아예 안됨, 256byte만 신뢰할 수 있음.(전송할 수 있는 데이터 양이 적다.)
속도는 post 방식보다 빠름

method = post방식 : 주소창에 전송하는 데이터들이 보이지 않는 방식
속도는 get방식보다 느림
보안은 get보다 좋음. 전송할 수 있는 데이터 양이 많다.

QueryString : ?mid=apple&mpwd=123456789&h1=hidden_data : 서버에 보내는 데이터
?로 시작, name=value 한 쌍으로 사용, name=value가 이어질 때는 & 사용
